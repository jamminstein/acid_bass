# acid_bass

> Acid bass generator with grid control and tap tempo

## Controls

- **E1** — tempo (BPM)
- **E2** — filter cutoff frequency
- **E3** — note release time
- **K2** — (unused)
- **K3** — (unused)

## Grid

8×16 layout: row 1 = tap tempo (col 1), generate (col 2); rows 2–8 control octave, gate length, swing, cutoff, accent probability, slide probability, and step mutes.

## Requirements

- norns
- grid (16x8)
- engine: PolyPerc (built-in)

## Install

```
;install https://github.com/jamminstein/acid_bass
```
